Bem vindo ao Web Server!

Algumas informações gerais sobre a utilização do Web Server na linha IndraDrive.

Introdução

Para a utilização do Web Server é obrigatório considerar no conjunto um display Advanced com cartão microSD de até 2Gb. A utilização do Web Server está ligada a criação de páginas HTML5 customizadas de acordo com a necessidade da aplicação que necessitam ser descarregadas ao Web Server através do IndraWorks. É possível realizar a leitura/escrita de parâmetros do drive e, inclusive, de variáveis presentes no MLD.

Funcionalidades

  • Páginas customizadas integradas internamente no Web Server do servo drive.
  • Comunicação com os parâmmetros e variáveis internas do servo drive via biblioteca ComLib.js.
  • Parametrização, diagnóstico, update de firmware sem a necessidade de nenhum hardware adicional.
  • Compatibilidade com documentos HTML5, CSS3 e Javascript.
  • Protocolo suportado: HTTP.

Highlights

Firmware

Para versões MPx18 e acima

A funcionalidade Web Server está implementada nas versões MPx18 e acima. Por padrão, se assim habilitado, o IMST (IndraMotion Service Tool) vem instalado no equipamento. Algumas funcionalidades já se encontram presentes ao tirar o equipamento da caixa.


IMST - IndraMotion Service Tool

Web Server padrão Rexroth

Upload de firmware. Backup rápido de parâmetros. Acesso fácil a funções de serviço. Diagnóstico fácil e rápido.


Páginas customizadas

Web Server para HTML5, CSS3 e Javascript

Compatibilidade com páginas customizadas, criadas pelo usuário e descarregadas ao servo drive via IndraWorks Ds. Assim, se torna uma IHM Web para funcionalidades básicas da aplicação, com acesso a todos os parâmetros do servo drive e a todas variáveis da lógica integrada MLD.
